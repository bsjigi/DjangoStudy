
# CRUD
    - 어떤 프로그램이던 간에 CRUD는 존재한다.
    - Create : 새로운 데이터를 생성하는 것
    - Read   : 이미 존재하는 데이터를 읽어오는 것
    - Update : 이미 존재하는 데이터를 수정하는 것
    - Delete : 이미 존재하는 데이터를 삭제하는 것

# 웹 서버의 CRUD
    - Create : 사용자(또는 관리자)가 보낸 정보를 DB에 저장하는것 (ex: 회원가입, 글 등록)
    - Read   : DB에 저장된 데이터들 중 원하는 데이터만 불러와 웹 페이지를 생성하여 응답하는것
               (ex: 마이 페이지, 글 목록, 상세 내용)
    - Update : DB에 저장된 데이터를 수정하는 것 (ex: 비밀번호 변경, 글 수정)
    - Delete : DB에 저장된 데이터를 삭제하는 것 (ex: 회원탈퇴, 글 삭제)
