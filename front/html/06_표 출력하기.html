# table 태그

- table: 표의 영역을 정의하는 태그
- tr: table row. 테이블에 한 줄을 추가하는 태그
- th: table header. 제목 칸을 추가하는 태그
- td: table data. 데이터 칸을 추가하는 태그
- caption: 테이블 제목을 작성하는 태그
