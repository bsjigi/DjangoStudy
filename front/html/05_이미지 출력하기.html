# img 태그

- src 속성 : 이미지의 경로를 적는다. 경로에 문제가 있으면 이미지가 나오지 않는다.
- alt 속성 : 이미지가 나오지 않을 때 나올 텍스트를 정의하는 속성
- width/height 속성 : 이미지의 너비/높이를 설정
- 너비와 높이 중 하나만 설정하면 원래 이미지 비율에 맞춰 다른 하나를 자동으로 수정한다
- 이미지 태그도 줄이 자동으로 바뀌지 않는 태그이다. (글자취급)
시각 장애인들을 위한 프로그램(스크린 리더)들은 이미지의 alt 속성에 정의된 값을 읽도록
설계되어 있기 때문에 alt속성을 반드시 채워주는 것이 좋다
