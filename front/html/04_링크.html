(중요) 링크를 클릭하는 것은, 서버로 요청을 보내는 것과 같다.
다음 행위들은 서버로 요청을 보낸다.
- 웹 브라우저 주소 창에 URL을 입력하고 Enter를 누르는 것
- 주소가 속성값으로 설정된 링크를 클릭하는 것

# 절대경로와 상대경로

  • 절대경로 : 드라이버의 시작지점부터 원하는 파일까지 모든 경로를 전부 입력하는 것
    절대 경로의 예
    1. D:\djangoStudy\front\html\04_링크.html
    2. D:\djangoStudy\django\my_first_project\manage.py
    3. https://comic.naver.com/webtoon/detail.nhn?titleId=662774&no=155&weekday=wed
    4. /webtoon/detail.nhn?titleId=662774&no=152&weekday=wed
  • 상대경로 : 현재 페이지(또는 파일)의 위치부터 원하는 파일을 찾아가기 시작하는 것
    상대 경로를 구분하는 방법 3가지
    1. ./ : 현재 폴더
    2. ../ : 상위 폴더
    3. 앞에 /가 없는 경우
