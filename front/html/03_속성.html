# 속성 (Attribute)

1. 모든 요소(element)는 속성을 가질 수 있다
2. 속성은 반드시 요소의 여는 태그에 정의해야 한다.
3. 속성을 통해 해당 요소에 대한 추가적인 정보를 브라우저에 제공할 수 있다
4. 속성은 대부분 속성 이름(name)과 속성 값(value)이 짝을 이룬다
5. 모든 요소에서 공통적으로 사용할 수 있는 글로벌 속성이 있다
6. 특정 태그에서만 사용할 수 있는 속성도 있다
